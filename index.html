Word wrapping only occurs when there is a word break. If you have a "word" that is as long as that, then there is no place for it to break. The proper solution is to write real content and not nonsense strings of characters. If you are using user generated content, then add a check for exceptionally long words and disallow them (or cut out part of them for URLs while keeping the whole thing in a link).
